Nesta página, você encontrará uma referência rápida a termos HTML comuns e uma tabela de entidades de caracteres nomeadas usadas com frequência.

Termos HTML

  
Atributo
  
Uma propriedade de elemento HTML que fornece informações adicionais ou funcionalidade para um elemento. Atributos aparecem como um par nome / valor com o valor contido entre aspas.
  
Elemento de nível de bloco
  
tags HTML que criam blocos de conteúdo dentro do documento, como cabeçalhos e parágrafos. Elementos de nível de bloco normalmente empilham um sobre o outro com base na ordem em que aparecem no código.
  
Elemento filho
  
Qualquer elemento HTML contido dentro de outro elemento HTML. Tags de elemento filho devem ser fechadas antes de suas tags de elemento pai.
  
Tag de fechamento
  
Aparece após o conteúdo de um elemento e indica o final de um elemento. É idêntico à tag de abertura de um elemento com a adição de uma barra (/) antes do caractere da tag.
  
Comentário
  
Informações de referência adicionadas ao código. Pode incluir uma visão geral de código mais complexo, um lembrete para futuros autores ou comunicações dentro de uma equipe.
  
CMS
  
O CMS é a abreviatura de Sistema de Gestão de Conteúdos e refere-se a qualquer sistema concebido para permitir aos utilizadores criar e manter conteúdo. Para a web, refere-se principalmente ao software projetado para gerenciar sites e seu conteúdo. Exemplos incluem o Joomla! e WordPress.
  
CSS
  
Cascading Style Sheets (CSS) é uma linguagem de apresentação usada para controlar a apresentação e o layout de páginas da web.
  
Doctype
  
Abrevia para Declaração de tipo de documento . Aparece no topo de todos os documentos HTML para acionar o modo de padrões e identifica a versão do HTML usada pelo documento.
  
DOM
  
O DOM, ou Document Object Model , é uma representação de documentos como o HTML. O DOM pode ser considerado como um esboço onde tags são representadas por nós. " Linguagens de script como o JavaScript fazem referência ao DOM ao acessar ou modificar o conteúdo da página.
  
Elementos
  
Os elementos HTML consistem em uma tag HTML, usada para identificar o conteúdo e o próprio conteúdo. É freqüentemente usado de forma intercambiável com os tags de termos, embora tecnicamente os tags se refiram apenas à tag em si.
  
FTP
  
FTP significa File Transfer Protocol . É um protocolo comum usado para transferir arquivos grandes de um servidor para outro.
  
HTTP
  
HTTP, ou Hypertext Transfer Protocol , controla a formatação e transmissão de páginas HTML em servidores web.
  
Elementos em nível in-line
  
Elementos que aparecem dentro do fluxo normal de texto. Ao contrário dos elementos de nível de bloco, os elementos de nível inline não iniciam novas linhas dentro do fluxo normal de documentos.
  
JavaScript
  
Linguagem de script desenvolvida pela Netscape e usada para controlar grande parte da funcionalidade e do comportamento na web.
  
Metadados
  
Informações adicionais sobre o documento ou conteúdo do documento, geralmente armazenadas em metatags.
  
Tipo MIME
  
Abreviação de Extensão de correio da Internet para múltiplos propósitos, Os tipos MIME são usados ​​para informar aos agentes do usuário que tipo de programa é um arquivo ou extensão. Sem um tipo MIME adequado, os agentes do usuário podem ter problemas para executar aplicativos ou processar arquivos corretamente.
  
Elemento pai
  
Um elemento HTML que contém outro elemento. É possível que um elemento seja filho de um elemento e pai de outro.
  
Elemento substituído
  
Qualquer elemento HTML em que o conteúdo do elemento é substituído por outra coisa. Exemplos seriam imagens, vídeos ou outro conteúdo de plug-in.
  
Tag
  
Caracteres de marcação que indicam o tipo de conteúdo exibido na página. A maioria das tags consiste em uma tag de abertura e fechamento.
  
URL
  
URL significa Uniform Resource Locator . Ele serve como o endereço de um documento ou outro recurso na web.
  
W3C
  
O W3C, ou o World Wide Web Consortium , é o corpo de padrões responsável pela criação da especificação HTML.
