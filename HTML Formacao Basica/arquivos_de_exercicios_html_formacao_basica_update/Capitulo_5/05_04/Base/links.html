Talvez o aspecto mais importante do HTML seja a capacidade de vincular um documento a outro. Este é o conceito básico subjacente à web e sem essa funcionalidade a World Wide Web não existiria.

Sintaxe de link

Criar links é relativamente simples, e a sintaxe fornece muita flexibilidade em onde e como os links são aplicados. Para criar um link, você usará o elemento âncora (⟨ a ⟩) para agrupar o conteúdo que deseja converter em um link. 

<a href="sintaxe.html" title="Aprenda mais sobre a sintaxedo HTML">Sintaxe do HTML</a>

Aqui, o texto "Sintaxe HTML" seria exibido como um link clicável. O atributo href diz ao navegador como resolver o link; é aí que o usuário deve ser direcionado quando o link é clicado. O atributo de título opcional fornece uma descrição do link e é útil para tornar o link mais acessível.

Tipos de ligação

Existem três tipos básicos de links: absoluto , relativo à raiz do site e documento relativo . Cada um desses tipos se refere ao valor do atributo href , que determina como o navegador deve resolver um link depois que o link for clicado.

Links absolutos

Esses links contêm o URL inteiro necessário para resolver um link, incluindo o protocolo. Isso geralmente é feito para links externos, que são links para páginas fora do site atual. Aqui está um exemplo de um link absoluto: 

 <a href="http://www.linkedin.com/learning" title="Você pode aprender com o LinkedIn Learning!">LinkedIn Learning</a>

Links relativos ao documento

Esses links são comumente usados ​​para navegar internamente em um site. Por exemplo, se você estivesse na página inicial de seu site e quisesse navegar até a página de contato, basta fornecer o caminho da página inicial para a página de contato para o valor de href . Isso exige que você entenda a estrutura de diretórios do seu site e quando você precisa navegar para dentro ou fora das pastas. Tome o seguinte exemplo: 

 <a href="contato.html" title="nossa página de contato">Entre em contato</a>

Este link pressupõe que a página de contato e a página atual estejam no mesmo diretório. Se a página de contato estivesse localizada em um diretório abaixo da página atual, o link ficaria assim: 

 <a href="recursos/contato.html" title="nossa página de contato">Entre em contato</a>

Isso pressupõe que a página de contato esteja em um diretório intitulado " resources " e é um nível abaixo da página atual. Para mover para fora de um diretório, você precede a página com o "../" personagens. Nesse caso do nosso link de contato, ficaria assim: 

 <a href="../contact.html" title="nossa página de contato">Entre em contato</a>

Esse link subirá do diretório atual e localizará a página contact.html no diretório pai. Você pode usar tantos desses "../" como você precisa subir e resolver adequadamente o link.

Links Relativos Site-Root

São semelhantes aos links relativos ao documento, mas começam na pasta raiz do site e depois passam pelos diretórios até a página desejada. Ao contrário dos links relativos a documentos, que dependem da localização atual da página para resolver adequadamente os links, os links relativos à raiz do site são gravados da mesma maneira, independentemente da localização atual em seu site. Todos os links são precedidos por uma barra (/) que se refere ao diretório raiz. Aqui está um exemplo: 

 <a href="/contato.htm" title="nossa página de contato">Entre em contato</a>

Neste exemplo, o link é resolvido indo diretamente para a pasta raiz do site e encontrando a página de contato. Se a página de contato estivesse localizada mais profundamente no site, o link ficaria assim: 

 <a href="/resources/contact.htm" title="nossa página de contato">Entre em contato</a>

Identificadores de fragmentos

Ocasionalmente, você deseja que um link se refira a um local específico em uma página. Isso pode ser feito usando um tipo especial de link chamado identificador de fragmento. Esses links apontam para elementos com um atributo ID específico. Digamos, por exemplo, que você tenha uma longa lista de FAQs em uma página organizada por categorias e tenha um título no topo de cada categoria. Se você atribuísse a cada título um atributo de ID, poderia usar esses IDs para criar links que saltassem para categorias específicas. Por exemplo, você poderia formatar o título Perguntas frequentes sobre campismo assim: 

<h2 id="acampamento">Acampamento</h2>

Em outro lugar da página, você pode vincular a essa seção criando o seguinte link: 

<a href="# camp">Saiba mais sobre o acampamento</a>

Observe o símbolo de hash (#) antes do valor de ID. Isso diz ao navegador para procurar um elemento com o valor de ID que se segue. Você também pode vincular a identificadores de fragmento em páginas externas. Basta anexar o identificador de fragmento ao URL e o link saltará não apenas para essa página, mas também para essa seção específica.
