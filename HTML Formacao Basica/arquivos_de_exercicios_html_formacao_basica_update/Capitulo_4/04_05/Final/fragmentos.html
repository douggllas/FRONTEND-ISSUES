Usando identificadores de fragmentos

Identificadores de fragmentos podem ser usados para vincular a outra seção dentro da mesma página ou a uma seção específica de outra página. Para usar identificadores de fragmento, você pode segmentar um elemento com um atributo de ID usando um link. Você resolve o link usando o valor do atributo ID precedido pelo hashtag (#).
