Definindo estilos

Uma maneira de visualizar como o HTML e o CSS trabalham juntos é imaginar um prédio em construção. Primeiro estrutura é construída e conforme o prédio sobe você não tem ideia de como ele vai parecer no final. Uma vez que a esrutura estiver pronta, o acabamento do edifício é adicionado. Isso pode ser tijolo, madeira, vidro ou outro materiial que mude a aparência externa do edifício.

O HTML e o CSS funcionam da mesma maneira. O HTML nos fornece a estrutura de nossas páginas. E o CSS nos da ferramentas para controlar como essa estrutura se parece, onde os elementos são posicionados e adicionar estilos decorativos adicionais. E mais, da mesma forma que um edifício pode mudar drasticamente, adicionando uma nova fachada, o design visual das páginas web pode ser alterado drascticamente simplesmente alterando o CSS da página.

Esta separação de estrutura e apresentação cria um fluxo de trabalho muito flexível e eficiente, onde a estrutura das páginas é independente de como as páginas são apresentadas. Isso permite que você atualize o estilo sem alterar o conteúdo da página e forneça diferentes designs visuais com base no contexto de onde a página está sendo exibida.
