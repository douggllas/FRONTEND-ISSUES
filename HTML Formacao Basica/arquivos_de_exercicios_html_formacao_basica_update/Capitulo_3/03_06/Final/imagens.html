Formatando o conteúdo da página

Nesta série de exercícios, exploraremos como usar elementos HTML para formatar o conteúdo básico da página.

Inserindo imagens

As imagens são colocadas na página através do uso da tag img . As imagens são consideradas conteúdo substituído , ou seja, o elemento é substituído pelo recurso que ele referencia. O elemento de imagem possui vários atributos importantes. O atributo src é usado para informar ao navegador onde encontrar a imagem. O atributo alt permite transmitir o texto descritivo que representa a imagem. Isso pode ser usado por leitores de tela ou exibido no caso de a imagem não carregar. As largura e altura são opcionais e podem ser usadas para passar as dimensões da imagem. para o navegador. Se omitido, o navegador ainda exibirá a imagem em sua largura e altura nativas. Esses valores podem ser alterados para forçar o navegador a dimensionar a imagem de maneira diferente do tamanho nativo.

A maioria dos navegadores suporta uma ampla gama de tipos de imagem. Para a maior parte, você está seguro usando os formatos de imagem jpg, png, gif, bmp e svg.
