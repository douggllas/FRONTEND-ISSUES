Aprendendo Web Design

"Web design "é um termo bem aberto. Muitas vezes as pessoas que são novas no web design sentem-se sobrecarregadas com tudo o que há para aprender. Isso é compreensível, mas lembre-se que é como qualquer outra disciplina;com dedicação e foco adequados, você progride rapidamente. Aqui está o meu conselho favorito para novos web designers no aprendizado de web design:

Aprenda como a web funciona.
Sempre me surpreendo com a quantidade de web designers que se concentram em aprender "exatamente o que precisam saber". e ignorar como a web realmente funciona. Sem entender como funcionam os servidores da Web, protocolos, navegadores e outras tecnologias da Web relacionadas, você nunca entenderá realmente o contexto por que você faz as coisas que você faz e o que faz uma prática ser melhor que outra.

Determine uma área de foco e comece por aí.
Existem muitas áreas de web design nas quais você pode se especializar, por isso recomendo aprender o máximo possível sobre elas, para determinar quais áreas interessam a você. Em seguida, passe tanto tempo aprendendo sobre sua área de foco específica quanto possível. Depois de atingir um nível de proficiência, comece a se expandir para outras áreas também.

Nunca pare de aprender. 
A web está em constante evolução. Mesmo se você se tornar o melhor absoluto que existe em um campo específico, é provável que surjam novas técnicas e abordagens durante o ano que tornarão menos desejável seu conjunto de habilidades atuais. Entenda que ser um bom web designer significa adotar a natureza mutável da web.

Quebre as coisas! 
Não há melhor professor do que experiência. Construa e experimente constantemente. Pergunte a si mesmo, "o que aconteceria se eu ..." ou "pergunto-me se eu poderia construir" ... " e depois sair e construí-lo. Se você é como eu, você fracassará uma boa parte do tempo, mas o processo de tentar as coisas e quebrá-las tornará você muito melhor. Sempre se desafie.
